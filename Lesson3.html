Lesson: Excel Basics

Learn the foundational skills of Microsoft Excel

Lesson Content

Microsoft Excel is a powerful spreadsheet application used for data analysis, organization, and automation. It’s widely used in business, finance, and academic fields.

In this lesson, you will learn about Excel’s interface, basic functions, formulas, formatting tools, and how to organize data efficiently.

Topics Covered:

  • Understanding Excel interface: cells, rows, columns, and sheets
  • Entering and formatting data
  • Basic formulas: =SUM(), =AVERAGE(), =IF()
  • Sorting and filtering data
  • Creating basic charts and graphs
